# Job Application - Terraform Outputs

Terraform configuration demonstrating infrastructure-as-code best practices.

<!-- BEGINNING OF PRE-COMMIT-TERRAFORM DOCS HOOK -->
## Requirements

| Name | Version |
|------|---------|
| terraform | >= 1.0 |

## Outputs

| Name | Description |
|------|-------------|
| applicant\_name | Name of the applicant |
| position\_title | Position being applied for |
<!-- END OF PRE-COMMIT-TERRAFORM DOCS HOOK -->

## Features

- GitHub Actions CI/CD pipeline with validation and security scanning
- Pre-commit hooks for code quality
- Input validation on variables
- Auto-generated documentation via terraform-docs

## Usage

```bash
terraform init
terraform plan
terraform apply
```
